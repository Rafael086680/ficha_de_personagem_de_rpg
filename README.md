# Ficha de Personagem RPG em HTML

Este projeto consiste em uma página HTML que representa uma ficha de personagem de RPG. O objetivo foi praticar a estrutura básica da linguagem HTML e a organização de conteúdo em uma página web.

## Objetivo

O projeto foi desenvolvido como exercício para praticar conceitos fundamentais de HTML, incluindo estrutura de página, organização de conteúdo e formatação de texto.

## Estrutura utilizada

Durante o desenvolvimento foram utilizados diversos elementos do HTML, entre eles:

- Títulos (`h1`, `h2`, `h3`)
- Parágrafos (`p`)
- Texto em **negrito**
- Texto em *itálico*
- Divisões de conteúdo utilizando `div`
- Inserção de imagens
- Links utilizando `href`

As `div` foram utilizadas para organizar melhor as seções da ficha do personagem, separando informações como atributos, descrição e outras características.

## Conteúdo da página

A página apresenta informações típicas de uma ficha de personagem de RPG, como:

- Nome do personagem
- Habilidades
- Atributos
- Mochila/Inventário
- História do personagem
- Imagem representando o personagem

## Tecnologias utilizadas

- HTML5

## Objetivo educacional

Este projeto foi desenvolvido com o objetivo de praticar e reforçar conceitos básicos de desenvolvimento web utilizando HTML.
